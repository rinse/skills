# Git Commit

コミットメッセージは一行で簡潔にしてください。

長い説明や複数行のメッセージは避け、変更内容を端的に伝える一行メッセージを書くこと。

**例:**

- `fix: resolve null pointer in user authentication`
- `feat: add CSV export to dashboard`
- `refactor: simplify database connection pooling`
